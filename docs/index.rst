===========================================
欢迎阅读 sCrypt IDE 官方文档
===========================================


**sCrypt IDE** 是一个强大的开发工具。通过该工具，可以轻松使用sCrypt语言来编写、编译、调试、测试、部署比特币合约，而无需安装其它软件。
它可用于专业开发，也适用于sCrypt智能合约的学习与教学。

**sCrypt IDE** 是一个 **Visual Studio Code** 插件 。在插件市场中搜索 sCrypt 就可以找到。
你可以以 `样板项目`_  为基础开始构建你自己的sCrypt项目。

本文档包含有关如何使用 **sCrypt IDE** 的说明。
